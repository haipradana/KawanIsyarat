BISINDO Kata Video Library
===========================

Letakkan video demo isyarat BISINDO di folder ini dengan nama:
  <kata>.mp4

Contoh:
  saya.mp4
  maaf.mp4
  terima_kasih.mp4
  tuli.mp4
  dengar.mp4
  rumah.mp4

Spesifikasi rekomendasi:
- Resolusi: 720x1280 (portrait) atau 1280x720 (landscape)
- Durasi: 2-5 detik (loop)
- Format: MP4 H.264
- Ukuran: < 2MB per video

Kalau file belum ada, `BisindoVideoPreview` akan fallback ke
placeholder ikon otomatis — tidak crash.
